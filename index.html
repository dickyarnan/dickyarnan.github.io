01

RELO'OZ

AWS LambdaAPI GatewayPostgreSQL RDSGitHub ActionsAWS SAM

Relo'OZ is a data-driven relocation platform that helps vulnerable individuals (single parents, the elderly, and people with disabilities) compare Greater Melbourne suburbs across safety, healthcare, transport, and housing. Backed by 13 integrated datasets from the ABS and Victorian Government, it turns complex relocation decisions into clear, accessible insights.

  • Designed and deployed 8 serverless Lambda functions as a complete REST API backend, each scoped to a single domain and declared in SAM template.yaml as Infrastructure as Code.
  • Built a GitHub Actions CI/CD pipeline for automated deployment on push, with S3-triggered Lambda for automated data ingestion and consistency.
  • Implemented multi-layer security including VPC private subnets, WAF rate limiting, and parameterised SQL queries to prevent injection by design.

BIRDEX

KotlinAndroidGoogle Maps APITFLiteRoom DBKotlin Flow

BirdEx is a citizen science Android app that helps Australian birdwatchers discover nearby hotspots and identify bird species through on-device AI sound recognition. It supports real-time wildlife conservation data collection aligned with SDG 15 (Life on Land).

  • Implemented a sensor simulator reading biometric and activity data from MIT-BIH and WISDM datasets via CSV, emitting through Kotlin Flow to trigger real-time alerts when user activity risks disturbing birds.
  • Developed the Quest screen for location-based hotspot discovery with dropdown filtering and bird species search across the app database.
  • Built Google Maps API integration for in-app navigation with route guidance, and OpenWeather API for weather forecasts at planned birdwatching locations.
